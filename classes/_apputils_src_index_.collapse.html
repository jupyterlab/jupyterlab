Class Collapse<T>

A panel that supports a collapsible header made from the widget's title. Clicking on the title expands or contracts the widget.

title

title: Title<Widget>

Inherited from IShell.title

  • Defined in node_modules/@lumino/widgets/lib/widget.d.ts:70

The title object for the widget.

Notes

The title object is used by some container widgets when displaying the widget alongside some title, such as a tab panel or side bar.

Since not all widgets will use the title, it is created on demand.

The owner property of the title is set to this widget.
